# wormhole3_account_bindings
Wormhole3 Account Bindings Contract on NEAR

## Environment Requirements

- Rust v1.66 (e.g. v1.66.1)
- Node v16 (e.g. v16.18.1)

## Set Up Project

```bash
npm i
```

## Build Contract

```bash
make build
```

## Test Contract

Run both unit test and integration test

```bash
make test
```
